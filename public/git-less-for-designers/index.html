Preprocessing CSS

CSS est contraignant

  • pas de variables
  • pas d'opérations
  • instructions différentes selon les navigateurs
  • paradoxalement peu d'héritage
  • pas de calcul sur les couleurs
  • beaucoup de répétitions

Bref, css est répétitif et pas facile à utiliser

(sans compter que les navigateurs/OS ne les interprètent pas de la même manière)

Less permet de résoudre une partie de ces problèmes

  • notamment avec les variables
  • et le calcul sur les couleurs

less est un programme qui

  • lit des instructions
  • depuis un fichier .less
  • les exporte vers un fichier .css (dit fichier compilé)

    • pas écrit directement
    • prêt à l'emploi pour le navigateur

Build Tetra

Un build Tetra

  • correspond à la génération de fichiers .less -> .css
  • différents .css sont agrégés dans un seul = build
  • reporté sur la webapp pour être déployé en prod
  • plusieurs builds sont générés, e.g. viadeo, apna, viadeo-rtl, minifié ou non
